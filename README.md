## Run Locally

Clone the project

```bash
  git clone https://github.com/abhi-2029/react-shopping-app
```

Go to the project directory

```bash
  cd react-shopping-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* [React](https://reactjs.org/)
* [Redux](https://redux.js.org/)
